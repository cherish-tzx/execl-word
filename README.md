# 文件对比工具

基于 Vue2 的文件对比模块，支持 Excel 和 Word 文件的精准对比。

## 功能特性

- 左右双栏文件上传对比
- 支持 Excel (.xls, .xlsx) 和 Word (.doc, .docx) 格式
- 保留原文件内容和样式
- 精准的字符级差异检测
- 相似度百分比显示
- 差异颜色标注：
  - 红色背景：删除的内容
  - 绿色背景：新增的内容
  - 橙色背景：修改的内容

## 安装依赖

```bash
npm install
```

## 运行项目

```bash
npm run dev
```

## 技术实现

- 使用 LCS（最长公共子序列）算法进行精准对比
- xlsx 库解析 Excel 文件
- mammoth 库解析 Word 文件
- 逐字符、逐单元格、逐行精准比对

## 使用说明

1. 点击左侧上传区域，选择第一个文件
2. 点击右侧上传区域，选择第二个文件
3. 系统自动进行对比并显示结果
4. 左侧显示原文件，右侧显示差异标注
5. 顶部显示文件相似度百分比
